# L'Essence - Luxury Perfume Store

An elegant, responsive Flutter e-commerce application for luxury fragrances. Designed with a clean, minimalist aesthetic suitable for high-end fashion and cosmetics brands.

## Features
- **Responsive Layout:** Adaptive grid layout scaling beautifully from mobile to desktop.
- **Product Catalog:** Mocked data representing a curated list of luxury perfumes.
- **Detailed Product Views:** Dedicated detail screen for each fragrance with descriptions and add-to-cart functionality.
- **Luxury Theme:** Minimalist typography, off-white backgrounds, and sharp contrasts.

## Tech Stack
- Flutter
- Dart
- Material 3

## Getting Started

1. Clone the repository
2. Ensure you have Flutter installed and configured
3. Run `flutter pub get`
4. Start the app via `flutter run`
